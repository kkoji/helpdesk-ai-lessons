ローカル環境の構築

前半のJupyterハンズオンをご自身のPCで進めるための、環境構築手順

Udemyラボ（Personal Plan / Business Pro限定）を利用する方は、このレクチャーの作業は不要です。
各ハンズオンのラボにはノートブック・データ・ライブラリが配置済みで、レクチャーを開くだけですぐに使えます。

このレクチャーを終えると、次から始まるハンズオンをすぐに動かせる状態になります。
ハンズオンやラボを進める中で行き詰まったときも、ここに戻ってきてください。

🧰
ここで整えるのは、いわば「道具箱」です。
作業用のフォルダを1つ作り、そこにPythonの仮想環境と、コース全体で使うライブラリをまとめてインストールします。
一度この環境を作ってしまえば、以降のハンズオンでは、レクチャーに添付されているノートブックをこのフォルダにダウンロードして開くだけで進められます。

事前準備：ターミナルと Python の確認

このコースでは Mac（macOS）／ Windows の両方に対応した手順を用意しています。
コマンドが異なる箇所は、その都度両方を併記しますので、ご自身の環境の方を実行してください。

ターミナル（コマンドを打つ画面）の開き方

OS 開き方
Mac 「ターミナル」アプリを起動（command + スペース で Spotlight を開き「ターミナル」と入力）。
Windows スタートメニューで「PowerShell」と入力して起動（コマンドプロンプトでも可）。

Python がインストールされているか確認

ターミナルで次を実行し、Python 3.10.x のようにバージョンが表示されればOKです（3.10 以上を推奨）。

Mac / Linux
python3 --version
Windows
python --version

表示されない・バージョンが古い（3.10 未満）場合は、下記の手順でインストールしてください。

Python のインストール（未インストールの場合）

1. python.org のダウンロードページを開く
2. お使いのOS（Windows / macOS）向けの最新版インストーラーをダウンロードする
3. ダウンロードしたインストーラーを実行してインストールする

⚠️Windowsの注意点：現行のインストーラーでは、基本的にデフォルトの設定のまま進めれば python コマンドが使えるようになります。
インストール後、python と打つとMicrosoft Storeが開いてしまう場合は、Pythonが正しくインストールされていないか、PATHが通っていない可能性があります。Pythonを入れ直すか、インストーラーの設定でPATHへの追加に関する項目を確認してください。
⚠️Macの注意点：新しいmacOSにはPythonが標準で入っていないことがあります（python3 --version が反応しない場合）。
その場合も上記の手順でインストールすればOKです。インストール後はターミナルを開き直してから再確認してください。

1作業用フォルダの作成

前半のハンズオンで使う専用フォルダを1つ作ります。
このフォルダに、以降ダウンロードするノートブックをすべてまとめていきます。

Mac / Windows 共通 
mkdir helpdesk-ai-lessons
cd helpdesk-ai-lessons
💡フォルダ名は一例です。
お好きな名前で構いませんが、GitHubリポジトリ名の helpdesk-ai（次のレクチャーで紹介）とは紛らわしいので別の名前にしておくことをおすすめします。

2仮想環境の作成と有効化

このフォルダ専用のライブラリ置き場（仮想環境）を作ります。
仮想環境を使うと、他のプロジェクトとライブラリが混ざりません。

仮想環境を作成

Mac / Linux
python3 -m venv .venv
Windows
python -m venv .venv

仮想環境を有効化

Mac / Linux
source .venv/bin/activate
Windows（PowerShell / コマンドプロンプト共通） 
.venv\Scripts\activate

有効化に成功すると、プロンプトの先頭に (.venv) と表示されます。以降のコマンドはこの状態で実行してください。

3コース全体で使うライブラリの一括インストール

このコースの前半で使うライブラリを、ここでまとめてインストールします。
ここで入れておけば、この先の各ハンズオンで追加インストールは基本的に不要です。

Mac / Windows 共通 
pip install jupyterlab ipykernel openai langchain-openai langgraph faiss-cpu numpy

少し時間がかかりますので、完了するまで待ちましょう。

インストールされる主なライブラリ

ライブラリ 用途
jupyterlab / ipykernel Jupyter Labの本体・ノートブック実行環境
openai OpenAI APIとの接続（生のAPI呼び出し）
langchain-openai LangChain経由でのOpenAI API接続
langgraph エージェントのグラフ構造
faiss-cpu ベクトル検索エンジン（FAISS）
numpy ベクトル演算

4Jupyter Labの起動確認

環境が整ったか、実際にJupyter Labを起動して確認しましょう。

Mac / Windows 共通 
jupyter lab

ブラウザでJupyter Labのランチャー画面が開けば、環境のセットアップは完了です。
確認できたら、いったんタブを閉じて構いません。

➡️次のハンズオンからの進め方：各ハンズオンのレクチャーには、そのレクチャー専用のノートブック（.ipynb）が添付ファイルとして用意されています。
ダウンロードした添付ファイルをこの helpdesk-ai-lessons フォルダに展開し、Jupyter Labの画面から開くだけで進められます（データファイルが必要なハンズオンでは、ノートブックと一緒に data フォルダも展開されます）。

OpenAI APIキーの設定

各ハンズオンのノートブックを実行すると、最初にAPIキーの入力を求めるプロンプトが表示されます。
入力した値は画面に表示されず、ノートブックにも保存されません。
APIキーの取得方法はレクチャー2-3で案内します。

ノートブック実行時のプロンプト 
OpenAI API キーを入力してください: ＿＿＿

環境変数として事前に設定しておく場合は以下のようにします。

Mac / Linux
export OPENAI_API_KEY="sk-..."
Windows（コマンドプロンプト） 
set OPENAI_API_KEY=sk-...
Windows（PowerShell） 
$env:OPENAI_API_KEY="sk-..."
💡この方法で設定した環境変数は、そのターミナルを閉じると消えます。
ターミナルを開き直したら再度設定するか、ノートブック実行時のプロンプトに直接入力してください。
⚠️APIキーをソースコードに直接書き込まないでください。
誤ってGitHubに公開してしまうリスクがあります。

よくある質問

python3 コマンドが見つからないと言われる（主に Windows）
Windows では通常 python（3なし）が正しいコマンドです。
python --version で確認してください。
それでも見つからない場合は、Python インストール時に「Add python.exe to PATH」にチェックが入っていない可能性があります。
py --version を試すか、Pythonを入れ直してください（Python 3.10以上を推奨）。
Windows で .venv\Scripts\activate を実行するとエラーになる（PowerShell）
PowerShell のセキュリティ設定（実行ポリシー）でスクリプト実行がブロックされている場合があります。
次のいずれかで解決できます。
・コマンドプロンプト（cmd） を使って .venv\Scripts\activate を実行する（最も簡単）。
・PowerShell を使い続けたい場合は、一度だけ次を実行して許可する：Set-ExecutionPolicy -Scope CurrentUser RemoteSigned（確認を求められたら Y）。
その後もう一度 activate を実行する。
pip install が失敗する / パッケージが見つからない
仮想環境の有効化を忘れていないか確認してください。
プロンプトの先頭に (.venv) が表示されていれば有効化されています。
表示されていない場合はステップ2の有効化コマンドをもう一度実行してください。
Jupyter Labがブラウザで開かない
ターミナルに表示されたURLを手動でブラウザに貼り付けてみてください（http://localhost:8888/... のような形式）。
ハンズオンのノートブックを開いたら、データファイルが見つからないエラーが出る
ノートブックと一緒に展開される data フォルダが、ノートブックと同じ階層に無いと発生します。
添付ファイルをダウンロードしたときの notebook/ ・data/ の配置を崩さずに展開してください。
